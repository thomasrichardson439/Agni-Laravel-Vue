# Agni

Sanofi Agni project. Used stack:<br>
- Laravel
- Vue.js
- Inertia.js
- Tailwind CSS


## Installation Instructions

`composer install`<br>
`npm install`<br>
`cp .env.example .env`<br>
Set all the things in env then <br>
`php artisan key:generate`<br>
`php artisan migrate --seed`<br>
`npm run watch` / `npm run dev`<br>
